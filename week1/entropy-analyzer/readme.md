# Entropy Analyzer

CLI tool that uses Shanon's Entropy to calculate entropy of an input vile and gives verdict on what kind of file has been provided.


# how to run it

go run main.go test_medium.txt 

# Example Ouput 

go run main.go test_medium.txt 
File: test_medium.txt
Entropy: 4.99
Verdict: LOW (plaintext likely)
